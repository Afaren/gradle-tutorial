Gradle循序渐进教程，同时也顺带推广Gradle让更多人认识并使用。

本人也是初学者，接触Gradle之前没有任何Groovy基础。

教程中包含了我的学习过程、**学习顺序（非常重要）**以及在过程中遇到的问题，相信对大多数初学者应该非常有帮助。另外，教程不会涉及到Gradle的所有方面，若有需要，请阅读官方文档。

文章中错误在所难免，如果发现，烦请提交issue通知我修改，希望大家一起学习进步。
